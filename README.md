# Hardening Checker

Linux system hardening checker.

Pemeriksa hardening sistem Linux.


## Description

Audits Linux systems against security best practices and hardening guidelines.

## Deskripsi

Mengaudit sistem Linux terhadap praktik terbaik keamanan dan panduan hardening.


## Installation

- Navigate to the project folder
- Run: chmod +x hardening_check.sh

## Instalasi

- Masuk ke folder proyek
- Jalankan: chmod +x hardening_check.sh


## Usage

- Audit: sudo ./hardening_check.sh
- Demo: ./hardening_check.sh --demo

## Penggunaan

- Audit: sudo ./hardening_check.sh
- Demo: ./hardening_check.sh --demo
